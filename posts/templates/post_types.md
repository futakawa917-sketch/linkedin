# 投稿タイプ定義

## 1. 専門性（expertise）— 40%
**目的**: 信頼構築。「この人はSNS運用のプロだ」と認知される
**構成**: フック → 課題提起 → ノウハウ（3-5ポイント） → まとめ → 軽いCTA（いいね/保存）
**例テーマ**:
- アルゴリズムの仕組み解説
- 投稿の構成テクニック
- KPI設計の考え方
- 業界別SNS活用の違い

## 2. 事例（case_study）— 30%
**目的**: 社会的証明。「実際に成果を出している」と信頼される
**構成**: フック（成果数字） → Before（課題） → 施策 → After（成果） → 学び → CTA
**注意**: クライアント名は伏せる。業界・規模・課題は具体的に
**例テーマ**:
- フォロワー少なくても問い合わせが来た事例
- 採用コスト削減に成功した事例
- BtoB企業のSNS活用事例

## 3. エンゲージメント（engagement）— 20%
**目的**: リーチ拡大。コメントを誘発してアルゴリズム評価を上げる
**構成**: 問いかけ → 自分の意見/体験 → 読者への質問
**例テーマ**:
- SNS運用あるある
- 業界の変化についての意見
- 「あなたならどうする？」系

## 4. CTA（cta）— 10%（月1-2回）
**目的**: リード獲得。DMでの相談を促す
**構成**: 価値提供（ノウハウ/事例）→ 「もっと詳しく知りたい方はDMください」
**注意**: 押し売り感を出さない。価値提供の延長として自然に
**例テーマ**:
- 無料SNS診断の案内
- 業界別SNS活用ガイドの提供
- 壁打ち・相談の案内
